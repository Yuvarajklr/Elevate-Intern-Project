### 4. ðŸš¨ Common Password Attacks

| Attack Type        | Description                                | Prevention                             |
|--------------------|--------------------------------------------|----------------------------------------|
| Brute Force        | Tries every possible combo                 | Long, complex passwords + rate limits  |
